# Data Parser 

Этот проект предоставляет пайплайн для обработки сырых CSV-данных резюме с HeadHunter,
извлечения ключевых признаков и сохранения их в формате NumPy (.npy) для дальнейшего анализа.
Пайплайн реализован с использованием паттерна "Цепочка ответственности" (Chain of Responsibility).
Проект фокусируется на создании робастного пайплайна для преобразования сырых данных в формат, пригодный 
для дальнейшего анализа и построения моделей машинного обучения.


 Основные этапы пайплайна:
1. Загрузка данных: Чтение CSV-файла с автоматическим определением разделителя и кодировки.
2. Извлечение признаков: Парсинг числовых и категориальных данных из текстовых полей резюме с использованием 
регулярных выражений и строковых операций.
3. Очистка: Обработка пропущенных значений, преобразование типов данных.
4. Сохранение: Результаты парсинга сохраняются в унифицированном формате .npy (массивы NumPy) для дальнейшей работы.

## Установка

Для запуска проекта требуются следующие библиотеки:
- pandas
- numpy

Установите их с помощью pip:

`pip install pandas numpy`

## Использование

Для запуска пайплайна используйте следующую команду, указав путь к вашему CSV-файлу:

`python app.py path/to/hh.csv`

После выполнения, в той же директории, что и исходный CSV-файл, будут созданы два файла:
- x_data.npy (матрица признаков: пол, возраст, опыт работы)
- y_data.npy (вектор целевой переменной: зарплата)

## Вывод

Созданный пайплайн способен преобразовать сырые данные в чистый и унифицированный формат для ML.
